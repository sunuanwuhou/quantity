# 语句排序-难

半做半猜

## 看首尾

1. 观察选项首尾句，**根据首尾句特点**，来排除一部分选项。

   + 适合首句

     > 1. 下定义： xx是xx
     > 2. 交代背景： 随着、近年来、目前、现在
     > 3. 比较： 与xxx相比

   + 不适合

     > 1. 出现代词，且指带不明
     > 2. 成对的关联词，后一个引导：  一方面、另一方面
     > 3. **分析问题不能前**： 提出问题、分析问题，提出在前

     

2. 尾句特点，慎用少用，可以不看

   > 表总结的，但是比较难判定。



## 对比选项，观察选项相同点和不同点，分析不同点

以下不分先后，遇到谁分析谁！

1. 代词： 就近指代   
2. 关联词： 成对出现
3. 时间顺序： 1936  1937 
4. 顺序词： 首先 然后 进而 
5. **局部高频词**： 134都出现一个词，134都围绕话题讨论，捆绑
6. 行文逻辑： 提出问题、分析问题、解决问题
