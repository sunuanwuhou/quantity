# 做题步骤

1. 边读边勾画**所有**的关联词、关键词、特殊标点，并根据其**暗示**来分析对应含义
   找到最优空

   + 最优空
     + 一空：没有空的句子往往有暗示点
     + 多空：优先最简单的、暗示最明显的 

2. 在原文中找到空运应填含义，暗示点，也就是上下文之间的逻辑关系

   1. 务必尽量用原文
   2. 尽量有句子
   3. 尽量全面

   > 公务员要依法行政！
