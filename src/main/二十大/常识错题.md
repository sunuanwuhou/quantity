# 习俗

+ 打春，是立春节气习俗之一。旧时人们在立春前一天用泥土做成春牛，立春日用红绿鞭抽打，因此该习俗活动称为“打春”。在民间有在立春日举行“鞭打春牛”仪式迎立春，祈福五谷丰登。
+ 踏青为春日郊游，也称“踏春”，
+ 碰到岁就是春节，碰到月圆就是中秋
