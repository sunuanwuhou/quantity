B.8.5     
C.8

```
这题有三种做法，1.看整体 2.看部分 3.十字交叉
1.--------------
售出80%的图书后，剩下的图书打折出售，结果所得利润比原计划少14%
如果全部都按打折呢 100%->少70%
那打折后的利润就是原来的 40%*(1-70%)=12%
假设定价100 计划140 打折112 112/140=80%

2.----------- 部分
0.2 打折少了14%
打折后的利润是 40%*(1-14%)=34.4 




3---------
40       8
   34.4     x=12 
x        2
112/140=80%
```
