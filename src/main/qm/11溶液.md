# 基本常识

+ 溶液=溶质+溶剂 【牢记，溶剂不能忘记加】

+ 浓度=溶质/溶液

  > 分母是溶液
  >
  > 分子是溶质

+ 溶质=溶液*浓度

+ 看到分子，分母想到相减，销售是相加

+ 定值思维：一般来说都是溶质不变，或者溶剂不变 

+ 一杯15%浓度的水，倒出相同比列后，还是15%

  > 2个对象混合成a，要想保持浓度不变，溶液的比值也要不变

+ 2个部分，混合一个整体，十字交叉

A.15 
B.18 
C.21 
D.24

```
常规做法，抓住溶质不变，设原来溶液为x
0.3x+6=（x+36）*0.25 求出x


-----十字交叉
A 30        
       1/4 = 5/3    得到溶液之比
B 6/36      
3f=36
5f=60  60*0.3=18 
```
