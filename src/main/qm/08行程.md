# 总结

+ 行程问题本质就是对S=VT 的运用，找到不变的量，进行比列
+ 进行真实值匹配，这里题目肯定会给出多个隐藏条件，需要自己推出来。

# 追击问题

+ 一般来说都是默认迎面相遇，追击相遇考的很少。
+ 同一点出发的，第二次追击后相差都是2S
+ 追击距离=（速度差）*时间  【主要是这个公式】
  + 什么是追击距离，就是最开始追的时候2个人相差的距离
+ 相遇就是 =（速度和）*时间

A.3.6 
B.20 
C.25

```
追击相差=速度差*时间  注意这里时间有坑
```
