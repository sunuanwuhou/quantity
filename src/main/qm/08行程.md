# 追击问题

+ 一般来说都是默认迎面相遇，追击相遇考的很少。
+ 同一点出发的，第二次追击后相差都是2S
+ 追击距离=（速度差）*时间  【主要是这个公式】

## 同一点出发的环形行程问题

+ 追击：速度差*时间

  S=(V车-V人)*t追
  S=V车*t发
  ------------------------------------

 S=(V车-V人)*t追
 S=V车*t发
