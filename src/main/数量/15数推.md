# 总结

+  循环2次才能说规律

+  相邻2项之间(不一定是挨着)存在大于3倍，优先考虑倍数

   +  正负号先忽略 

+  单调数列：相邻2项都在3倍以内，优先考虑和差分析

   +  构造网格：和上一级有什么关系

+  九宫格： 看行看列

+ 7项以上，优先考虑分组
  + 间隔分组  a b c  d e f    【ace def】
  + 22分组  【ab】【 cd】【 ef】
  + 33分组  【abc 】【def】
  
+ 分数数列 【分子分母可以同比列放缩，没必要全部看】
  + 分子分母单独看
  + 分子分母四则运算与下一项的关系
  
+ 拆分：每项都是三位数

  + 四则运算看规律
