# 三不看原则

2. 不看百分号
3. 不看0

```
1856/3167
3167=3.167=3+1/6=19/6

2911/2348
2911/2.348=2+1/3=7/3
291*(3/7)

```

```
11/5=110/15=22/3
分子加0是为了约分
分母加小数点是为了变为乘法好计算
```

# 算误差

```
1.截三位做差
2.差除以(1,2) 估算首位 就是百分之几

2356/1032 
103-100=3
3/10=首位是0.3 就是3%

```

要保证误差在10%以内，这个需要练习，才能保证精确度！！！
