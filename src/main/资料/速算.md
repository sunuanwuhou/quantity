# 划线法

1. 误差：分母方向与分母一致，分子相反
2. 同向抵消、异向叠加
3. 计算误差 看增量/差量与前2位，比较大小，小就是10%一下，反之。【但是10%尽量不要出现，会影响口算】
4. 计算结果 160 上升2%  是3.2 ，163.2
5. 如果快速计算，找倍数关系、约分、化简1  【这里比较麻烦，有的一眼看不出来】
