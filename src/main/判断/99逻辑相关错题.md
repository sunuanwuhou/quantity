李：或者甲班所有学生都爱学英语，或者乙班所有学生都爱英语

A．甲班所有学生爱学英语，乙班所有学生都爱学英语

## 直言命题

（1）只有小红报名，小白、小黑和小花才会都跟着报名； 

（2）如果小白不报名，则小黑也不报名； 

（3）如果小黑不报名，则小灰也不报名； 

（4）小红没报名； 

（5）小灰报名了。 

由此可以推出（  ）。

A．小白、小黑和小花都报名了

B．小白和小黑都报名了

C．小黑和小花都报名了

D．小白和小花都报名了

```
(-小白 或 -小黑 -或 -小花 ) 或 小红

小白 小黑为真 
或命题 否一推一   -小花 为真 选B
```
