# 黑白运算

网上规律

整体: 对称、移动、叠加

部分： 部分、个数、相邻比较



+ 格子多 >=9  优先考虑

  + 【串起来黑白】 【 特征：黑色相连】【一笔画成不是一笔画】

  + 对称  【白色对称】【黑色对称】

    > 残缺的九宫格/16/25 基本都是考对称

  + 相邻比较 【变或不变】

  + 数黑的个数

+ 格子少 <=9  优先考虑

  + 平移:  【黑色个数基本不变】
  + 黑白叠加
  
+ 残缺916 优先考虑对称
